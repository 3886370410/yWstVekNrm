## 核心功能
- 用户登录注册
- 社团信息展示与管理
- 社团活动发布与管理
- 问卷调查创建与管理
- 公告信息发布与查看
- 留言反馈提交与查看
- 个人信息查看与编辑

- 社团活动管理
- 社团成员管理

- 活动计划管理
- 问卷调查管理

- 问卷调查与反馈收集
